# フォーマット共通ルール

全ツイート種別で適用。スキル固有の上書きがある場合はそちらを優先。

## 文字数

| 種別 | 制限 |
|------|------|
| 短文ツイート | 140字以内 |
| 引用RT | 100字以内 |
| 長文ツイート | 制限なし（250-350字目安） |
| BIPツイート | 140字以内 |

### X Premium文字数制限とタイムライン表示

- X Premiumプレミアムプランでは最大25,000字まで投稿可能（ベーシック: 4,000字）
- 長文ツイートの文字数制限は緩いが**タイムライン上では280字で折りたたまれ「もっと見る」表示**になる
- **最初の280字で興味を引く構成にする**ことが必須。数字・インパクトのある事実は280字以内に配置する
- 「もっと見る」の先には背景解説・利用方法・個人見解など「読み進めたくなる」補足情報を配置

## 読点（、）

- **短文ツイート / 引用RT**: 完全禁止。文の区切りは改行のみ
- **長文ツイート**: 最小限の使用は許容。基本は改行で代用

## 絶対禁止（例外なし）

- 絵文字 0個
- ハッシュタグ禁止
- デフォルトはcasual（温和な常体で統一）。politeモード指定時は敬語可（tone.jsonのformality_modes参照）
- 伸ばし棒語尾禁止（「〜だよなー」「〜かなー」）

## 改行ルール

- 1つの文・文章塊の途中での改行は完全禁止
- 段落間の改行はOK（意味的に異なる内容・話題の区切り時）
- 短文を頻繁に改行しない（「Aか / Bか / Cだ」はインフルエンサー風で作為的）
- 読点・接続詞で自然に繋ぐ

## 引用リンク

- ツイート本文の直下に必ず引用リンクを配置
- URLがない場合: 情報源を明示（「なし（[情報源の説明]）」）

## 略語禁止

サービス名・サイト名は正式名称で表記:
- ❌「HN」→ ✅「Hacker News」
- ❌「PH」→ ✅「Product Hunt」
- ❌「GH」→ ✅「GitHub」
