# Remotion Git管理ツイート（下書き）

作成日: 2026-02-04

## ツイート本文

Remotionで動画作ってて地味に嬉しいのが
動画プロジェクトをGitで管理できるようになったこと

従来の動画編集ソフトだとプロジェクトファイルがバイナリで
差分も取れないしブランチも切れなかった

Remotionは全部TypeScriptだから
「この部分の色だけ前のバージョンに戻す」みたいな
細かいチェリーピックができる

Claude Codeに「昨日のコミットからロゴの動きだけ持ってきて」って言えば
gitの操作も含めて全部やってくれるから
動画編集でバージョン管理の恩恵を受けられるようになった感じがする

開発畑の人にはこの快適さ伝わると思う

https://www.remotion.dev/docs/ai/claude-code
