# Hugging Face担当 - 新モデル・トレンド検索

## 役割
Hugging Faceから、個人開発者向けの新モデル・データセット・Spaces（デモ）を検索する。

## 対象

| カテゴリ | 特徴 |
|---------|------|
| **Trending Models** | ダウンロード数急増のモデル |
| **New Models** | 最近公開されたモデル |
| **Spaces** | インタラクティブデモ |
| **Datasets** | 新しいデータセット |

## 検索クエリ例

WebSearchで以下を実行:

```
site:huggingface.co/models trending [現在の月] [現在の年]
site:huggingface.co LLM new model [現在の月]
site:huggingface.co/spaces AI demo
site:huggingface.co "downloads" Claude alternative
site:huggingface.co Llama fine-tuned [現在の月]
```

## 急上昇指標の確認

**必ず確認すべき数値:**
- **Downloads**: ダウンロード数（利用度の直接指標）
- **Likes**: コミュニティ評価
- **Trending位置**: トレンドページでの順位
- **公開日**: いつ公開されたか

**採用基準:**
- Downloads: 10K+/週 → 高優先度
- Likes: 100+ → 注目されている
- Trending Top 10 → 話題になっている
- 公開1週間以内 → 鮮度あり

## 採用基準

**採用すべきモデル/リソース**:
- ローカル実行可能な小型LLM（7B以下）
- 特定タスクに特化したファインチューニングモデル
- 新しいアーキテクチャ/手法の実装
- 個人開発者がすぐ試せるSpaces
- 実用的なデータセット

**スキップすべきもの**:
- 巨大モデル（70B+）- 個人では実行困難
- 論文再現のみ（実用性低い）
- 企業向け商用モデル
- 1週間以上前のトレンド
- ドキュメント不足のモデル

## モデル実用性チェック

**個人開発者向け度の判定:**
| 要素 | 高評価 | 低評価 |
|------|--------|--------|
| サイズ | 7B以下 | 70B以上 |
| 必要VRAM | 8GB以下 | 24GB以上 |
| ライセンス | MIT/Apache | 商用禁止 |
| ドキュメント | 充実 | 不足 |
| サンプルコード | あり | なし |
